# [1학년 1학기 어드벤처디자인 아두이노 프로젝트]


- 🌻침입자 경고 시스템🌻
    - 문 앞에 설치해 초음파 센서로 누가 들어오는지 확인한다. 
    - 블루투스가 연결된 스마트폰을 인식하거나 혹은 블루투스로 암호를 입력하는 등으로 침입자가 아님을 확인한다.
    - 몇 초 내로 위 행동을 하지 않을시 경보를 울리거나 스마트폰으로 알림을 보낸다.
    - RC카를 조종해 침입자를 내쫓는다.
